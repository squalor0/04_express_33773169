About This Project

This website was created as part of my Dynamic Web Applications labwork at Goldsmiths University.

It uses Node.js and the Express framework to demonstrate server-side concepts such as routing, and dynamic content generation.

About Me

My name is Blake Murphy. I enjoy working with backend technologies, exploring how servers handle requests, and learning to build efficient web applications.

Project Features

  • Multiple routes serving different HTML pages
  • Dynamic route that displays user input (/welcome/:name)
  • Chained route handlers demonstrating next()
